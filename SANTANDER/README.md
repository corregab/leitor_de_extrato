# Santander extractor

This folder contains the Santander-specific extractor and instructions.

Use the root README for general info; run the extractor from this folder or call it with full path.

Example:

```powershell
python SANTANDER\income_extractor.py "../Extrato consolidado mensal (3) (6).pdf" --amounts-only --decimal-comma --out valores_virgula.txt
```
